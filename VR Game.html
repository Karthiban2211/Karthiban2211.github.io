VR Game Developer Intern


Overview :

I joined this professional project mid-development, taking over as the sole developer to lead the technical roadmap through to completion. My role involved a mix of systems engineering and content creation—stabilizing the inherited codebase while simultaneously building out new gameplay features, mission structures, and 3D environments. While the project was built for an immersive platform, the core development focused on high-performance PC-based architecture

Technical Contributions:

Systems Refactoring & Architecture: Audited the inherited codebase and overhauled the interaction logic to a more modular structure. This allowed for faster feature implementation and significantly reduced technical debt.

Feature Engineering & Mission Design: Designed and implemented several new gameplay features and mission-critical systems from scratch. I developed the logic for objective tracking, progression triggers, and interactive world events to expand the game’s scope.

Technical Art Pipeline: Managed the end-to-end integration of 3D assets. I was responsible for importing, configuring, and optimizing 3D models and environments, ensuring that high visual fidelity did not compromise the game's performance.

Performance Optimization: Utilized the Unity Profiler to identify and eliminate CPU bottlenecks and memory leaks. Successfully maintained a consistent 72+ FPS by optimizing draw calls and physics calculations—critical for both VR and high-end PC gaming.

Project Management: As the only developer, I acted as the technical bridge for the entire team. I managed the Git repository, established the version control workflow, and handled the final technical implementation of all art and sound assets.

Version Control & Workflow: Managed the Git repository and established a clean branching strategy. I served as the primary technical point of contact for the art and design teams, ensuring a smooth integration pipeline in an Agile environment.
